Privacy Policy

This tool analyzes public URLs to assist with Google Search Console integration. No personal or identifiable data is collected or stored.

1. No Data Collection

No names, email addresses, IPs, or identifiers are collected. This tool simply encodes provided URLs.

2. Public Page Processing Only

Only publicly available URLs are used to generate encoded values compatible with the GSC API.

3. No Tracking or Cookies

We do not use cookies, local storage, or any persistent tracking technologies.

4. Hosting and Security

This tool is hosted securely on Vercel infrastructure. No sensitive input is retained after processing.

5. Contact

If you have any privacy concerns, please contact the developer directly via your GPT interface or project repository.
